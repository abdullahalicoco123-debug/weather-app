# 🌦️ Weather App

A modern and responsive Weather App built with **HTML5**, **CSS3**, and **JavaScript**. The application allows users to search for any city and instantly view real-time weather information using a Weather API.

🔗 **Live Demo:**  
https://abdullahalicoco123-debug.github.io/weather-app/

🔗 **GitHub Repository:**  
https://github.com/abdullahalicoco123-debug/weather-app

---

# ✨ Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature
- 💧 Humidity information
- 💨 Wind speed display
- ☀️ Weather icon
- ⚡ Instant weather updates
- ⌨️ Search using button or Enter key
- 📱 Responsive design
- 🎨 Clean and modern interface

# 💻 How It Works

1. Enter a city name in the search box.
2. Click the **Search** button or press **Enter**.
3. The application sends a request to the Weather API.
4. The API returns real-time weather information.
5. The app displays the city's weather details instantly.
